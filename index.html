About Gaba

A Journey Into Language Design

Gaba is a hobby project where I'm building my own programming language from scratch. Instead of only using existing languages, I wanted to understand how they are built—from converting source code into tokens, parsing them into an Abstract Syntax Tree (AST), and finally interpreting and executing the program.

This project is not intended to compete with existing languages. It is a learning journey and an opportunity to experiment with language design, compiler concepts, and runtime environments while gradually adding features found in modern programming languages.

Getting Started

Installation Guide

Install Gaba on Linux, macOS, or Windows using a single command.

Linux & macOS

Run this command in your terminal to install the latest version:

curl -fsSL https://raw.githubusercontent.com/DivanshLpu/gaba/main/install.sh | bash

Uninstall: curl -fsSL https://raw.githubusercontent.com/DivanshLpu/gaba/main/uninstall.sh | bash
Path: ~/.local/bin/gaba

Windows (PowerShell)

Open PowerShell as Administrator and run:

irm https://raw.githubusercontent.com/DivanshLpu/gaba/main/install.ps1 | iex

Uninstall: irm https://raw.githubusercontent.com/DivanshLpu/gaba/main/uninstall.ps1 | iex
Path: %LOCALAPPDATA%\Programs\gaba\gaba.exe

Windows (Command Prompt)

Open Command Prompt and run:

curl -fsSL https://raw.githubusercontent.com/DivanshLpu/gaba/main/install.bat -o install.bat && install.bat

Uninstall: curl -fsSL https://raw.githubusercontent.com/DivanshLpu/gaba/main/uninstall.bat -o uninstall.bat && uninstall.bat

Verify & Troubleshoot

Verify installation by running:

gaba version

Troubleshooting

  • Linux/macOS: If gaba is not found, add this to ~/.bashrc or ~/.zshrc:
    export PATH="$HOME/.local/bin:$PATH" then run source ~/.bashrc
  • Windows: If gaba is not recognized, simply restart your terminal or open a new Command Prompt/PowerShell window.

                    What's in Gaba
                

            


            

                

                    

                    
Control Flow

                    
Full support for if/else, while, and for loops, plus stop and
                        skip controls.

                


                

                    

                    
Functions

                    
Define with fun, return values with give. Supports nested calls and recursion.

                


                

                    

                    
Lists & Maps

                    
Built-in data structures with methods like
                        ~push(), ~keys(),
                        indexing, and assignment.

                


                

                    

                    
Modules

                    
Import standard libraries like import "math" and call methods cleanly.

                


                

                    

                    
Native Functions

                    
Built-ins include say(), ask(), and type
                        converters like int(), str(), bool().

                


                

                    

                    
Interpreter Pipeline

                    
Learn from the clean architecture: Lexer →
                        Parser (AST) → Interpreter Runtime.
